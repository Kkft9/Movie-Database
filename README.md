* Created the interface using React and NodeJS

## Getting Started ##

* Install dependencies
```
    npm install
```

* Install nodemon
```
    npm install -g nodemon
```

* Run backend server
```
    cd omdb-backend
    nodemon start
```

* Run frontend
```
    cd omdb-ui
    npm run start
```
Now browse to the app at [http://localhost:3000](http://localhost:3000)
